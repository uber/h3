Bindings

As a C library, bindings can be made to call H3 functions from different programming languages. This page lists different bindings currently available. Contributions to this list are welcome, please feel free to open a pull request.

Athena​

  • daniel-cortez-stevenson/aws-athena-udfs-h3

Azure Data Explorer​

  • Functions for Working with H3 Indexes

BigQuery​

  • CARTO Analytics Toolbox

C#​

  • entrepreneur-interet-general/h3.standard
  • richardvasquez/h3net - A translation instead of a binding
  • pocketken/H3.net - A translation instead of a binding

ClickHouse​

  • Functions for Working with H3 Indexes

Clojure​

  • Factual/geo

Dart​

  • festelo/h3_flutter
  • codewithsam110g/h3_flutter_plus

Databricks​

  • H3 geospatial functions

Delphi​

  • SkybuckFlying/h3_delphi

DuckDB​

  • isaacbrodsky/h3-duckdb

ECL​

  • hpcc-systems/HPCC-Platform

Erlang​

  • helium/erlang-h3

Go​

  • uber/h3-go

Java​

  • uber/h3-java

  • realPrimoh/h3-reactnative

Kinetica​
