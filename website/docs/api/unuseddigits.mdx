These functions allow modifying the unused indexing digits in H3 indexes. For example, a resolution 13 cell index
will have 6 bits at the end of the index for the resolution 14 and 15 indexing digits that are set to all 1s for
a valid index. These functions allow storing data in those bits, which creates indexes with additional data embedded
in them at the trade-off of the indexes being invalid.

Valid indexes can be recreated by setting the modified index's unused digits to the value from `getMaxUnusedDigits`
for the index's resolution.

## getMaxUnusedDigits

Returns the maximum value that can be provided to setUnusedDigits
for a resolution.

This value is the same as the value that should be set for the index to be
a valid cell.

## getUnusedDigits

Retrieve the value stored in the unused digits of the H3 index.

For valid cells this will always be the value of getMaxUnusedDigits for
the cell's resolution.

## setUnusedDigits

Creates an index with the unused digits set to another value.

Unless data is the value provided by getMaxUnusedDigits, this will create
an invalid cell. To create a valid cell again, pass the created index in
to this function with `data` set to the value given by getMaxUnusedDigits.
